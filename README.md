# Help Desk Security Incident Ticketing Lab

## Overview
This lab simulates a Tier 1 Help Desk technician handling a security-related support ticket involving suspected malware activity on a user workstation.

## Skills Demonstrated
- Security ticket documentation
- Incident triage and classification
- Windows Defender alert review
- Windows Event Viewer (Security logs)
- Tier 1 escalation procedures

## Scenario
A user reports frequent pop-ups, browser redirects, and system slowness after visiting a non-work-related website.

## Actions Taken
- Documented user-reported symptoms
- Reviewed Windows Defender protection history
- Checked Windows Security Event Logs
- Classified incident as suspected endpoint malware
- Escalated to Tier 2/Security team per policy

## Tools Used
- Windows 10/11
- Windows Defender
- Windows Event Viewer
- Notepad

## Outcome
The issue was documented and escalated appropriately following Tier 1 Help Desk security procedures.
